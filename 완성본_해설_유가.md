## 2. 파트별 해부 (각 파트 = 질문 1개 + 방법 + 실제 결과)

### 📈 Part 1. 연도별 가격 — "4년간 어떻게 변했나?"
- **방법**: 평균·중앙값·표준편차·IQR·왜도(기초통계) + 선그래프(평균 ±1표준편차 밴드)
- **실제 결과**:

| 연도 | 평균가격 | 표준편차 | 비고 |
|---|---|---|---|
| 2018 | 1,609원 | 57.0 | |
| 2019 | 1,535원 | 55.8 | |
| 2020 | **1,284원** | 65.2 | 🦠 코로나로 역대 최저 |
| 2021 | 1,559원 | 51.8 | 다시 반등 |

- **발견**: 기름값 쌀 때(2020)는 주유소 간 격차도 줄어듦. 왜도(+)라서 "비싼 주유소 몇 개"가 평균을 끌어올림 → **중앙값**이 더 현실적.

### 🏷️ Part 2. 브랜드 — "어디가 비싸고 잘나가나?"
- **방법**: **ANOVA**(3개 이상 그룹 차이 검정) + η²(효과크기) + Boxplot + Stacked Bar/Heatmap(점유율)
- **실제 결과**: F=80.5, p=1.46e-114(<0.001) → "브랜드별 가격차이는 우연 아님" ✅
  - η²=**0.048** → 브랜드가 가격차이의 약 4.8% 설명 (작은 효과)
  - **알뜰계열이 제일 쌈**, 자가상표가 제일 비싸고 분산 큼
- **점유율 변화**: SK에너지 31.7%→27.2%(📉 하락) / 현대오일뱅크·알뜰계열 소폭 상승

### ⛽ Part 3. 셀프 vs 일반 — "셀프가 진짜 싼가?"
- **방법**: **독립표본 t-검정(Welch)** + Cohen's d(효과크기) + Paired dot plot
- **실제 결과**: 셀프가 **매년 더 쌌고** 4년 평균 **약 –28원/L**

| 연도 | 셀프평균 | 일반평균 | 차이 | Cohen's d |
|---|---|---|---|---|
| 2018 | 1,595 | 1,619 | –23원 | –0.43 |
| 2019 | 1,519 | 1,546 | –27원 | –0.51 |
| 2020 | 1,264 | 1,297 | **–34원** | –0.56 |
| 2021 | 1,541 | 1,571 | –29원 | –0.62 |

- **소비자 의미**: 1년 720L 기준 **연 약 2만원 절약**. 2020년 저유가 때 격차가 가장 큼(일반이 더 천천히 내림).

### 📍 Part 4. 지역 — "어디 살면 제일 비싸게 넣나?"
- **방법**: 지역별 기초통계 + **분산분해**(지역간/지역내) + 수평 Boxplot/히트맵 + Folium 지도
- **실제 결과(2021)**:

| 순위 | 지역 | 평균 | 특징 |
|---|---|---|---|
| 1 | **서울** | 1,641원 | 제일 비쌈 + 편차도 최대(std 152) |
| 2 | 제주 | 1,593원 | 비싸지만 거의 균일 |
| … | 강원·충남·경기 | 1,564~1,565원 | |
| 최저권 | 부산·경남·대구 | ~1,543원 | 싼 편 |

- **발견**: 지역이 가격차이의 **약 15%** 설명(브랜드보다 영향 큼). 기름값 쌀 때도 지역 격차 유지 → 임대료·운영비 구조 반영.

### 📊 Part 5. 가격 안정성 — "가격을 자주 바꾸나?"
- **방법**: **변동계수(CV) = 표준편차 ÷ 평균 × 100** + 산점도/추세선 + Violin
- **실제 결과**: 대부분 **CV < 1%** → 일주일간 가격 거의 안 바꿈 (전체 평균 0.268%)
  - 가장 안정적: 알뜰(ex) 0.157% / 가장 변동 큼: 알뜰주유소 0.326%

### 🧩 Part 6. 종합 결론
- 꼭대기 결론 + 근거 4개를 한 장으로 묶고 → **이해관계자별 시사점**:

| 이해관계자 | 시사점 |
|---|---|
| 소비자 | 셀프+알뜰 선택으로 연 수만 원 절감 |
| 정유사 | 알뜰 확대 대응, 브랜드 로열티 전략 재검토 |
| 정책 | 서울·제주 프리미엄은 규제보다 경쟁구조 개선으로 |
| 창업자 | 알뜰 가맹 vs 고마진 지역 입점 양극화 |

## 4. ⚠️ 그대로 베끼면 안 되는 이유 (완성본의 숫자 불일치)
발표 문장(글)과 실제 계산된 숫자가 **안 맞는 곳**이 있다. 글은 따로 쓰고 숫자는 코드가 계산했는데 둘을 안 맞춘 흔적.

| 항목 | 글/슬라이드 주장 | 실제 계산값 | 판정 |
|---|---|---|---|
| 브랜드 효과크기 η² | "≈0.10, 큰 효과" | **0.048, 작은 효과** | ❌ 불일치 |
| 셀프 효과크기 Cohen's d | "≈0.2, 작은 효과" | **평균 0.53, 중간 효과** | ❌ 불일치 |
| 브랜드 ANOVA F | "F > 100" | **F = 80.5** | ❌ 불일치 |
| 연도 평균가격 | 1,268 / 1,553 | 1,284 / 1,559 | ⚠️ 단위 차이(미세) |

👉 **직접 숫자를 다시 계산해 글과 맞추면 모범답안보다 더 정확한 결과물**이 된다.

## 6. 알아야 할 통계 용어 (한 줄 정리)
| 용어 | 쉽게 | 어디 쓰였나 |
|---|---|---|
| 평균/중앙값 | 보통값 / 한가운데값 | 연도·지역 가격 |
| 표준편차·IQR | 값이 흩어진 정도 | 가격 격차 |
| 왜도(Skewness) | 분포가 한쪽으로 쏠림 | 고가 주유소 영향 |
| ANOVA + η² | 3집단↑ 차이검정 + 그 차이가 얼마나 큰지 | 브랜드 |
| t-검정 + Cohen's d | 2집단 차이검정 + 차이 크기 | 셀프 vs 일반 |
| 분산분해 | 차이가 "그룹 간"인지 "그룹 안"인지 쪼갬 | 지역 |
| 변동계수(CV) | 평균 대비 들쭉날쭉함 | 가격 안정성 |
