# 문제 정의 — 서울 지하철 혼잡도 분석

## 1. 배경
서울 지하철은 하루 수백만 명이 이용하는 핵심 대중교통. 혼잡은 단순 불편을 넘어 **안전(압사 위험)** 문제이며, 코로나로 급감했던 이용객이 빠르게 회복 중.

## 2. 문제 상황
코로나 이후 수요가 **2024년 기준 코로나 이전(2019)의 약 91%까지 회복**되며 **혼잡이 재심화**. 그러나 혼잡은 전 노선에 고르게 퍼진 게 아니라 **특정 호선·역·시간대에 극심하게 집중**(상위 10% 역이 전체 이용객의 **28.5%**)되어, **한정된 예산·인력을 어디에 투입할지** 불분명하다.

## 3. 분석 질문 (가설)
1. 수요는 얼마나 회복됐나? (재혼잡 여부)
2. 어느 호선·역이 가장 혼잡하고 얼마나 집중되나?
3. 혼잡은 언제(시간대) 발생하며, 유입형(업무)·유출형(주거) 역은?
4. 혼잡 역은 지리적으로 어디에 몰리나?

## 4. 분석 대상 / 활용 (투트랙)
- 🚇 **서울교통공사** (운영·실행): 안전·운영효율·비용
- 🏛️ **국회 국토교통위원회** (입법·예산): 제도·재원·시민편익
> 정당색 없이 **초당적(안전·효율·예산)** 논리로 작성.

## 5. 기대 효과
데이터 근거로 **"집중 지점 우선 투자 + 수요 분산 제도"** 정책 제안·실행계획(Action Plan) 도출.

## 6. 분석 범위
- 데이터: 서울 지하철 호선별·역별·시간대별 승하차 (2015~2026, 28노선·600역)
- 추이 = 전체 기간 / 혼잡·시간대·역별 = 최근월 단면
