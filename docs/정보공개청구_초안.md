## ✅ 제출 절차 (직접)
1. open.go.kr 로그인 → **정보공개청구 → 일반청구**
2. 위 **제목·청구내용** 붙여넣기
3. **청구기관 찾기 → 서울교통공사** 선택
4. **청구인 정보 입력**(본인) — 직접
5. **개인정보 수집·이용 동의** 체크 — 직접
6. **청구 접수(신청)** 버튼 — 직접 (※ 제출은 본인이)

- 기간(3년)이 부담되면 **최근 1년**으로 줄여도 OK.
