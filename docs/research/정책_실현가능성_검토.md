# 정책 실현가능성·우선순위 검토 (팀원 피드백 반영)

> 팀원이 제기한 비판(노출 보정·포퓰리즘·실현성·예산 우선순위·미래수요)을 공개 데이터로 검토. 모든 수치 출처 명기, 추정은 '추정' 표기.

## 0. 사고 "출근 2.65배" — 이용객(노출) 보정 (팀 데이터 직접 계산)
| 08시 (시간평균 대비) | 배수 |
|---|---|
| 사고 건수 | ×2.65 |
| 이용객 수 | ×1.79 |
| **노출 보정(백만 명당 사고율)** | **×1.66** |
| **포아송 회귀(출근 vs 비출근)** | **×1.27** |
| 출근(7~9) vs 낮 비피크(11~16) | ×1.12 (≈ 비슷) |

- **결론**: "출근에 사고가 1인당 더 자주 난다"는 약함(건수 多 = 사람 多). **핵심 메시지 = "사고가 출근 피크(≈4명/㎡ 밀집)에 겹치면 피해·혼란 최대" = 겹침의 영향.** (출처: 서울교통공사 5년 사고현황 + 서울시 시간대별 승하차 자료, 팀 계산)

## 1. 정시율·운행장애 벌과금
- 한국 99%대는 **UIC 16분 기준**. **5분 기준은 88.9%**(2023, 코레일), KTX 79.1%로 하락 중. 서울교통공사는 **정시율을 공식 공표하지 않음**(투명성 문제). (데일리안 2024 / 지표누리)
- 홍콩 MTR: 5분 기준 99.9%, 운행중단 시 벌과금→승객 환급(2025.5 HK$1,920만). (SCMP·HKFP 2025)
- **판단**: 벌과금 직이식은 곤란(단일 민간운영자·요금조정 구조 없음) → **'5분 기준 정시율 공표 + 운행중단 승객 환급제'로 변형**.

## 2. 예산 — '신규 지출' 아닌 '이미 편성→가속'
- 노후 재투자 **4조 6,543억(전동차 3.85조 포함), 2027까지 진행 중**. (경향 2023)
- 증차 4·7·9호선 **1,025억, 국비 256억(25%) 확보, 진행 중**. (리얼타임데일리 2024)
- 그러나 **누적적자 ~19.7조, 무임손실이 순손실의 54.3%** → 긴축으로 **5호선 신호개량 순연 위기**. (경향·이투데이 2025 / 전기신문 2025)
- **판단**: 제안 = "이미 약속된 안전사업을 적자·긴축에 밀리지 않게 **제때 집행** + 무임손실 **국비 분담**으로 여력 확보". '새 돈 쓰자'보다 근거상 강함.

## 3. 미래 수요 — 피크 투자 유효한가
- 생산연령인구 2022 3,674만 → 2052 2,380만 (**-35.2%**). (통계청 장래인구추계 2023)
- 그러나 **수도권 집중은 2034년까지 지속**, 재택 잠식 제한적(주중 통행 -5.8%). (통계청·국토연구원)
- **판단**: **향후 10년 서울권 피크 투자 유효**. 단 20년 시계엔 '대규모 증차'보다 **가동률·유연 투자**가 합리적.

## 4. 시차출퇴근·요금 정책 — 효과 작음
- 시차/장려금 실측 분산 효과 **~2%**(2026, '모두의카드'); 증차(집중배차)가 더 즉효(사당 -10%p). (news1·뉴스핌 2026)
- 의무화 비현실: 현행 **30% 권고**, 공무원 유연근무 이용률 **17.7%**. (news1 2026)
- 대중교통 차등요금 효과 작음: 멜버른 첨두 **-1.2~1.5%**, **서울 조조할인(2015) 분산 0.07% 실패**. (Infrastructure Victoria / 경향 2015)
- 첨두 수요 **비탄력(-0.24)**·역진성 → **'혼잡세' 할증은 부담 큼**. (VTPI / NBER 2023)
- **판단**: 시차·요금은 **보조수단**. 할증(혼잡통행료·혼잡세·연동요금) **보류**, '비혼잡 할인'은 효과 작음 인지하고 보조로만.

## 5. 정책 실현가능성 판정표
| 정책수단 | 실현성 | 핵심 근거 |
|---|---|---|
| 안전투자·노후교체 **가속** | **상** | 4.65조 이미 편성, 긴축으로 순연 위기 → '가속' |
| 혼잡완화 **증차** | 상(10년)/중(20년) | 1,025억 진행·국비25%; 수도권 집중 2034까지 |
| 동선개선·집중배차·비상대응 | **상** | 저비용·기존 권한·즉효(사당 -10%p) |
| 정시율 공표+승객 환급제 | 중 | 측정 불투명 개선, 비용≈0 |

## 발표 시 사실관계 주의 3
1. "한국 정시율 99%라 벌과금 불필요" ✗ → 그건 16분 기준, 5분 기준 80%대.
2. "증차에 새 예산" ✗ → "이미 편성된 4.65조 제때 집행 + 무임손실 국비분담".
3. "도로 혼잡통행료 성공 → 지하철도" ✗ → 통근객 시간이동 어려움(조조할인 0.07%).
