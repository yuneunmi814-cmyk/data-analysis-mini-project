# 데이터 수집 정의 — 서울 지하철 혼잡도 분석

## 분석 범위 (확정)
- **노선**: 서울 **1~9호선** (수도권 광역·코레일 노선 제외 / 9호선 연장구간은 9호선에 병합)
- **시점**: 혼잡 단면 = **최근월(2026-05)** · 추이 = **전체기간(2015~2026)**
- **"혼잡" 정의**: **역 승하차 인원(유동인구) 기준** — 열차 내 혼잡도(%)는 선택 보강
- **기준 역 수**: 271개 역(1~9호선, 최근월)

## ⚠️ 핵심 구분
| 의미 | 데이터 | 사용 |
|---|---|---|
| 역 유동인구 = 승하차 인원 | ✅ 보유 | **메인** ("어느 역이 붐비나") |
| 열차 내 혼잡도(%) | 별도(서울교통공사) | 선택 보강 |
